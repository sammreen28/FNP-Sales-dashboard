# Sales Performance Dashboard

## Overview
This project analyzes sales data using Orders, Customers, and Products datasets to identify key insights on revenue, customer behavior, and product performance.

## Key Insights
- Identified top-performing occasions and categories driving revenue  
- Analyzed customer spending and order patterns  
- Highlighted best-selling products and high-demand cities  
- Evaluated delivery time to find improvement opportunities  
- Tracked monthly sales trends and peak order periods  

## Tools Used
- Excel-Pivot table, power query, power pilot
